# musica
vamos hablar de la musica 
La música es una forma de expresión artística que usa sonidos y silencios organizados en el tiempo para transmitir emociones, ideas o sensaciones.

Se basa principalmente en:

Ritmo: el pulso o compás que marca el tiempo.

Melodía: la sucesión de notas que se pueden “tararear”.

Armonía: cómo se combinan varios sonidos a la vez.

Timbre: el “color” del sonido (no suena igual una guitarra que un piano).

La música puede servir para expresarse, relajarse, celebrar, acompañar historias, bailar o simplemente disfrutar. Está presente en casi todas las culturas del mundo y en muchos momentos de la vida diari
